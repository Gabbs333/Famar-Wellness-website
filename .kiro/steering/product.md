# Product Overview

## Famar Wellness - Professional Website & CMS

Famar Wellness is a professional website with a comprehensive content management system (CMS) for a wellness/health business. The project includes both a public-facing website and an admin dashboard for content management.

### Core Features

**Public Website:**
- Service showcase and descriptions
- Technology stack display
- About section
- Gallery with image optimization
- Testimonials
- Booking system with Calendly integration
- Blog/News section
- Contact forms

**Admin Dashboard:**
- Authentication system
- Content management (pages, posts, media)
- Blog category and tag management
- Contact and booking management
- Media library with image optimization
- Analytics dashboard
- Settings management
- Rich text editor (Tiptap) with component library
- Template system for page layouts

### Key Differentiators
- Built-in image optimization and intelligent compression
- Template-based page editor with change propagation
- Performance-focused architecture
- Supabase backend with PostgreSQL database
- TypeScript for type safety
- Tailwind CSS for styling

### Target Audience
- Wellness/health professionals
- Business owners needing a professional online presence
- Content managers requiring easy-to-use CMS

### Languages
- Primary: French (public-facing content)
- Admin interface: Mixed French/English
- Codebase: English
